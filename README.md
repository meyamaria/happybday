***Birthday Celebration***

## Features
Firework Display 
Animated Birthday Message
Candle Flame Animation
Click anywhere to trigger a firework explosion.
Responsive Design

## To Use 
1. Download / Clone the repo
2. Download firework audio(Pixabay) and save it in the same folder
   (Or replace the `<source src="...">` tag with your own audio URL)
3. pic.html` in a browser
4. Click to interact and enjoy the animated celebration!

>>>  Make sure the user interacts (clicks) with the page before audio plays due to browser autoplay restrictions.

## File Structure

project/

->pic.html           # Main HTML file with embedded script

->fireworks-sound.mp3  # Firework sound effect (you can replace this).

## Technologies
* HTML5 Canvas for graphics
* JavaScript for animation
* CSS for styling
* Google Fonts: *Gloria Hallelujah*
